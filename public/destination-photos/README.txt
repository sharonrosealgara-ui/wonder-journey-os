🌴 WONDER JOURNEY — REAL DESTINATION PHOTOS
============================================

Drop a real photo here, named after the place, and it appears
automatically on the Travel Passport. No code editing needed!

HOW:
1. Save a photo as a .jpg
2. Name it exactly the place's id (all lowercase), for example:
      manila.jpg
      banaue.jpg
      palawan.jpg
      iloilo.jpg
      bago.jpg
      guimaras.jpg
      boracay.jpg
      cebu.jpg
      bohol.jpg
      siargao.jpg
      mayon.jpg
      philippines.jpg
3. Put it in this folder (public/destination-photos/)
4. Commit + push (or drag-drop deploy) — it shows up live!

If no photo is here for a place, the app shows its emoji art instead,
so nothing ever looks broken.

TIP: landscape photos (wider than tall) look best. Keep files under ~500 KB.
Use your own photos, or free ones from Wikimedia Commons / Unsplash /
Pexels / the Philippine Department of Tourism.
