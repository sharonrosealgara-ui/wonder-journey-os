🌴 WONDER JOURNEY — REAL LESSON PHOTOS
=======================================

Drop a real photo here, named after the lesson, and it appears
automatically at the top of the lesson. No code editing needed!

HOW:
1. Save a photo as a .jpg
2. Name it exactly the lesson's id (all lowercase, with dashes), e.g.:
      welcome-to-the-philippines.jpg
      tagalog-hiligaynon-greetings.jpg
      filipino-family-values.jpg
      mango-float-adventure.jpg
      national-symbols.jpg
      turon-adventure.jpg
      meet-teacher-sharon.jpg
      where-i-live.jpg
      daily-life-philippines.jpg
3. Put it in this folder (public/lesson-photos/)
4. Commit + push (or drag-drop deploy) — it shows up live!

If no photo is here, the app shows the lesson's emoji art instead,
so nothing ever looks broken.

TIP: landscape photos look best. Keep files under ~500 KB.
